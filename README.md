## MemoryCard-PC_Game 
### ゲーム紹介
このゲームは、ゲームジャムみたいの活動中に開発されました<br>
これは日本の神経衰弱みたいのフロップゲームです<br>
普通な神経衰弱と最も異なる点は<br>
中のカードはすべて台湾のスナックと食べ物です<br>
一部のカードは特に高いスコアを持っています<br>
裏返すと、特殊効果や効果音もあります<br>
