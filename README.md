# Memory-Game
這個遊戲是類似日本神經衰弱的翻牌遊戲<br>
裡面的卡片皆為台灣特色的小吃及食物<br>
與普通神經衰弱最不同的點在於
有五種食物的分數特別高
翻開時也會有特別的特效及音效<br>
